# 🐉 Python Animation

An interactive  cursor animation inspired by procedural motion and creative coding techniques. Built using HTML5 Canvas, CSS, and JavaScript, this project creates a smooth dragon-like creature that dynamically follows the user’s cursor in real time.

The animation uses connected body segments and motion interpolation to generate realistic snake-like movement with fluid trailing effects and fantasy-inspired visuals.
